Privacy Policy

Last updated: May 2026

HouseCheck ("we", "us") operates the website at housecheck.it.com. This policy explains what data we collect when you use the site and how it is handled.

Summary We do not create accounts, track individuals, or sell data. We receive postcodes you search and standard server log data. No cookies are set by us.

What data we collect

Search queries (postcodes). When you search for a postcode, it is sent to our API as a URL query parameter. We do not store search queries in a database. The query is used only to retrieve and return price data from our dataset.

Server logs. Our API hosting provider (Google Cloud Run) automatically records standard server log data for each request, including your IP address, the request URL, timestamp, and response status. These logs are retained for up to 30 days and are used solely for monitoring and debugging.

Website hosting logs. Our website is hosted on Vercel, which may log IP addresses and request metadata. See Vercel's privacy policy.

Cookies

We do not set any cookies. No tracking, advertising, or analytics cookies are used on this site.

Third-party services we load (Google Fonts and jsDelivr CDN for charting) may set their own cookies or log your IP address subject to their own policies. These are used only to deliver fonts and chart functionality.

Third-party services

  • Google Fonts — fonts are loaded from Google's CDN. See Google's privacy policy.
  • jsDelivr — Chart.js is loaded from jsDelivr's CDN. See jsDelivr's privacy policy.
  • Google Cloud Run — our API backend is hosted on Google Cloud Run. See Google Cloud's privacy notice.

Data source

All house price data is sourced from HM Land Registry Price Paid Data, published under the Open Government Licence v3.0. HM Land Registry is not responsible for the accuracy of this site.

Legal basis for processing

We process data under legitimate interests (Article 6(1)(f) UK GDPR) — specifically, operating and maintaining a functional web service. Server logs are necessary to detect errors and prevent abuse.

Your rights under UK GDPR

You have the right to:

  • Access personal data we hold about you
  • Request erasure of your personal data
  • Restrict or object to processing
  • Lodge a complaint with the Information Commissioner's Office (ICO)

Because we do not store search queries or create user profiles, we hold minimal personal data (server logs only, retained 30 days). Requests for access or erasure can be made by contacting us below.

Contact

For privacy-related enquiries, email: housecheck.it@gmail.com

Changes to this policy

We may update this policy from time to time. The "last updated" date at the top of this page will reflect any changes.
